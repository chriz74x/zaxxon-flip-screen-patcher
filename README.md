# Zaxxon Arcade Flip Screen Patcher

Online tool to patch Sega Zaxxon ROMs (U27/U29) for a 180° flip screen in Upright Mode.

## How to use
1. Dump your original ROM1 and ROM3.
2. Upload them to the web tool.
3. Download the patched ZIP and flash to new EPROMs.
